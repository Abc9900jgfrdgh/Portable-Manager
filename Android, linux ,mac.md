## üì• Installation (Windows)
1. **Download** the `Controller.bat` file from this repository.
2. **Download** `rclone.exe` from the [official site](https://rclone.org/downloads/) (Windows version).
3. Put both files in the **same folder** (e.g., inside a folder named `DriveManager` on your USB stick).
4. Run `Controller.bat`.

---
## üõ† Features
* **True Portability:** Works from any folder path (USB, External SSD, Desktop).
* **Auto-Mounting:** Automatically finds free drive letters (P:, Q:, R:...) to avoid conflicts.
* **Encryption Made Easy:** Creates secure, encrypted folders on Google Drive or Local Storage without complex commands.
* **Smart Detection:** Detects existing Rclone processes to prevent "drive letter collisions" if you run multiple instances.

## üåç Cross-Platform Compatibility (Android & Linux)
This tool is designed to generate **"Portable Configs"**. This means you do not need to edit the config file when moving between operating systems.

### üì± How to use on Android
1. Run the tool on Windows and create your drives (Google Drive, Encrypted Local, etc.).
2. Locate the `.conf` file in your USB folder (e.g., `gdrive_1234.conf`).
3. Transfer this file to your phone.
4. Install **RCX** or **Round Sync** (Rclone clients) from the Play Store.
5. Go to **Settings > Import Config** and select your file.
6. **Success!** Your Encrypted Drive opens immediately. No setup required.
