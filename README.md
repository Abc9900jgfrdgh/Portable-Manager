# Rclone Portable Manager

A robust, portable Windows batch script to manage Rclone mounts, encryption, and config files on the go.

## Features
* **True Portability:** Runs from any USB drive or folder.
* **Auto-Mount:** Detects available drive letters (P: to Z:) and avoids collisions.
* **Multi-Volume:** Supports config files with multiple drives.
* **Encryption:** Easily creates encrypted folders on Google Drive or Local USB.
* **Safe Exit:** Wipes RAM, Clipboard, and Cache on exit (keeps configs safe).

## How to Use
1.  Download **`Controller.bat`** from this repository.
2.  Download **`rclone.exe`** from [rclone.org](https://rclone.org/downloads/).
3.  Put both files in the **same folder** (e.g., on your USB stick).
4.  Run `Controller.bat`.
